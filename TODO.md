# Order System — План развития проекта

## Текущее состояние (MVP завершен)

### Реализовано

#### Domain

* User
* Customer
* Product
* Order

#### Repository

* CustomerRepository

  * Add
  * GetByID

* OrderRepository

  * Add
  * GetByID

* Memory реализации репозиториев

#### Service

* DiscountCalculator

  * VIP скидка 10%

* OrderService

  * CreateOrder

#### Main

* Создание клиента
* Создание товара
* Создание заказа через сервис
* Вывод результата

## Что важно помнить

Главная цель проекта — не количество функций.

Главная цель проекта:

* понять интерфейсы
* понять сервисы
* понять репозитории
* понять dependency injection
* научиться разделять бизнес-логику и хранение данных

Проект считается успешным уже сейчас, потому что MVP завершен.
Дальнейшие этапы нужны для обучения и закрепления архитектурных подходов.
